# Skypro Wallet

Учебный проект электронного кошелька на React. Приложение позволяет зарегистрироваться, войти в аккаунт, вести список расходов и смотреть аналитику по категориям за выбранный период.

## Технологии

- React
- Vite
- React Router
- styled-components
- Recharts
- react-icons
- Fetch API
- Skypro Wallet API
- localStorage для хранения токена авторизации

## Что реализовано

- регистрация пользователя через API;
- вход в аккаунт через API;
- сохранение токена в `localStorage`;
- защита страниц `/expenses` и `/analytics`;
- выход из аккаунта с удалением токена;
- получение списка расходов из API;
- добавление нового расхода;
- удаление расхода;
- валидация форм входа, регистрации и добавления расхода;
- выбор даты расхода через стандартный календарь браузера;
- календарь выбора периода на странице аналитики;
- динамическая диаграмма расходов по категориям;
- фильтрация аналитики по выбранному периоду;
- адаптивная верстка страниц авторизации, регистрации, расходов и аналитики.

## Сценарий ручной проверки

1. Зарегистрировать нового пользователя.
2. Войти под созданным пользователем.
3. Перейти на страницу `Мои расходы`.
4. Добавить новый расход.
5. Удалить расход.
6. Обновить страницу и проверить, что данные загрузились из API.
7. Перейти на страницу `Анализ расходов`.
8. Выбрать дату или период в календаре.
9. Проверить, что сумма и диаграмма меняются по выбранному периоду.
10. Нажать `Выйти` и убедиться, что защищенные страницы больше не открываются без входа.

## Работа с датой

В форме добавления расхода используется стандартное поле браузера `type="date"`. Перед отправкой в API дата преобразуется в формат `ДД.ММ.ГГГГ`, который используется внутри проекта для отображения и фильтрации.

## Хранение данных

Основные данные пользователей и расходов хранятся на стороне API. В `localStorage` приложение сохраняет только:

- `skypro_wallet_token` — токен авторизации;
- `skypro_wallet_current_user` — данные текущего пользователя.

При выходе из аккаунта эти данные удаляются.
