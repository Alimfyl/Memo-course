# Skypro Wallet

## Описание

Skypro Wallet — приложение для учета личных расходов.

Пользователь может:

* зарегистрироваться;
* войти в систему;
* просматривать список расходов;
* добавлять новый расход;
* анализировать расходы по категориям.

---

## Стек технологий

* React
* Vite
* React Router DOM
* Styled Components

---

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5173
```
